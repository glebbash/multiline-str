multiline

  • multiline(strings: TemplateStringsArray, ...args: unknown[]): string
    • Defined in multiline-str.ts:21

    Strip indentation of multiline strings based on length of the last line.
    Notice: first and last lines will be removed.

    example
    console.log(multiline`
        line 1
        line 2
        line 3
        `)

    Prints:

    line 1
line 2
line 3

    Parameters

    • strings: TemplateStringsArray

      Array of template strings.

    • Rest ...args: unknown[]

      Template string arguments.

    Returns string
